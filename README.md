# 进程调度算法

## 操作系统

### 问题
编写一个进程调度算法，使得系统能够在多道程序环境下，合理地分配系统资源，最大限度地提高系统的吞吐率和响应时间

### 解决方案

|   3    |      2       |      5       |
|   4    |      3       |      2       |
|   5    |      4       |      4       |
